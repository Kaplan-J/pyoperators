h1. {{ page.title }}

The current version of the pyoperators package is {{ site.version_stable }}, and it is still in beta stage.

h2. The easy way

The easiest way to download and install the pyoperators package is by using the easy_install utility:

bc. sudo easy_install pyoperators

h2. The Git way

The project is hosted at "github":https://github.com/pchanial/pyoperators/ you can download it by cloning the project, using the version control system "Git":http://git-scm.com/:

bc. $ git clone git://github.com/pchanial/pyoperators.git

This method is convenient to test pyoperators' development branch. The package can be updated by typing (if you've just cloned the repository, you will obviously not get anything new):

bc. $ cd pyoperators.git
$ git pull
Already up-to-date.
