What's new ?
* New operators To1dOperator, ToNdOperator for 1-d to N-d indexing conversions.
* SymmetricBandToeplitzOperator, useful to represent stationary (inverse) covariance matrices.
* Add helpers assert_is_type, assert_same, complex_dtype_for, izip_broadcast

Under the hood:
* Simplify alignment handling
* API change: Wavelet2Operator -> Wavelet2dOperator
